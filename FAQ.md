# ❓ Frequently Asked Questions (FAQ)

This document answers common questions about the Receipt Manager project.

## 🚀 General Questions

### What is Receipt Manager?
Receipt Manager is a full-stack web application designed to simplify receipt management and expense splitting. It uses AI to scan receipts and extract line items.

### Is it free to use?
Yes, the project is open-source and free to use. However, you will need to set up your own Firebase project and Google Cloud project, which may have their own costs.

### Which AI model does it use?
The project uses Google's **Gemini 3 Flash Preview** model for receipt parsing and chat commands.

## 🛠 Technical Questions

### How do I set up the project?
Please follow the instructions in the **README.md** file.

### Which technologies are used?
The project is built with React, TypeScript, Vite, Tailwind CSS, Firebase, and Google APIs.

### How do I run the tests?
Run `npm test` to run all tests.

### Can I contribute to the project?
Yes, we welcome contributions! Please follow the instructions in the **CONTRIBUTING.md** file.

## 🔒 Security Questions

### Is my data secure?
Yes, we follow security best practices and use Firebase security rules to protect your data. Please read the **SECURITY.md** file for more details.

### How do I report a security vulnerability?
Please report vulnerabilities privately by emailing the project maintainers at [your-email@example.com].

## ☁️ Cloud Services Questions

### How do I connect my Google Drive?
The app will prompt you to connect your Google Drive when you first use it. You can also manage your connection in the settings.

### How do I export to Google Sheets?
The app will automatically export your purchases to a Google Sheet if the "Sync to Sheets" option is enabled in the settings.
